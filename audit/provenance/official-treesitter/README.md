Snapshots of the OFFICIAL tree-sitter grammar highlight queries
(tree-sitter/tree-sitter-<lang>, fetched 2026-07-20). These serve as the
"fifth witness" in provenance scoring (see docs/PHILOSOPHY.md): an ontology
of which distinctions the modern consensus recognizes - weighty on
*existence* of a distinction, silent on *color*, and not inerrant (the
capture vocabulary is uneven across grammars, and the JS query ships the
same SCREAMING-case @constructor heuristic that misclassifies class names).
